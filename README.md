Clean setup to start a new project

### Tools ###

* Webpack
* Pug
* Sass
* Babel 
* Normalize.css
* <i>... more to come </i>

### Run

* npm install
* npm run start - work with livereload on dev server
* npm run build-pretty - build your project without minified files
* npm run build - build your project with minifed files
